# EPUB Reader Chrome Extension

一个功能完整的Chrome浏览器扩展，用于在浏览器中阅读EPUB格式电子书。

## 功能特性

- 📚 **双栏布局**：左侧目录面板，右侧内容阅读面板
- 📋 **完整目录**：解析并显示EPUB的目录结构，支持多级目录
- 🔄 **目录折叠/展开**：支持目录的折叠和展开功能
- 🎯 **章节导航**：点击目录项快速跳转到对应章节
- 📁 **文件导入**：支持拖放或文件选择器导入EPUB文件
- 📝 **保持格式**：保持EPUB原有的文本格式和样式
- 🖼️ **图片支持**：正确显示EPUB中的图片资源
- 📱 **兼容性**：支持EPUB 2.0和EPUB 3.0格式

## 安装方法

1. 下载或克隆此项目
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹
6. 扩展图标将出现在浏览器工具栏中

## 使用方法

1. 点击浏览器工具栏中的扩展图标
2. 在弹出的窗口中，拖放EPUB文件或点击按钮选择文件
3. 阅读界面会显示：
   - 左侧：目录面板，可点击章节导航
   - 右侧：内容阅读区域
4. 使用目录面板顶部的"收起"按钮可以隐藏目录
5. 点击"打开新文件"可以切换书籍

## 技术实现

- 使用原生HTML5、CSS3和JavaScript开发
- 使用JSZip库解析EPUB文件（EPUB本质是ZIP压缩包）
- 支持EPUB 2.0（NCX目录）和EPUB 3.0（NAV目录）
- Chrome Manifest V3标准
- 无外部依赖（除JSZip CDN）

## 注意事项

- 请确保使用合法获得的EPUB文件
- 扩展仅在本地处理文件，不会上传任何内容
- 对于非常大的EPUB文件，可能需要稍等片刻加载
